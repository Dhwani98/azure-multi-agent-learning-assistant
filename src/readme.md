# Azure Multi-Agent Learning Assistant

A personalized learning assistant using:
- Azure OpenAI
- RAG (FAISS-based retrieval)
- Semantic Kernel (learning path planning)

## Features
- Interactive chat-based learning
- Retrieval-augmented responses
- Personalized learning path
- Persistent session memory

## Setup

```bash
git clone <repo>
cd Azure_multi_agent_learning_assistant
python -m venv azure_project_env
azure_project_env\Scripts\activate
pip install -r requirements.txt
